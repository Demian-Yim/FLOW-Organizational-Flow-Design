# 📊 FLOW Admin 대시보드 사용 설명서

**FLOW Admin**은 교육 문의 및 진단 데이터를 실시간으로 관리하고 분석할 수 있는 통합 관리 도구입니다.

---

## 1. 접속 및 로그인
- **주소**: `[앱 URL]/admin`
- **방법**: Firebase Authentication에 등록된 관리자 계정(이메일/비밀번호)으로 로그인합니다.
- **보안**: 권한이 없는 사용자는 자동으로 로그인 페이지로 리다이렉트됩니다.

## 2. 주요 기능 안내

### 🏠 대시보드 홈 (Overview)
- **핵심 지표**: 총 문의 수, 총 진단 수, 전환율을 실시간으로 요약하여 보여줍니다.
- **주간 추이**: 최근 7일간의 데이터 유입량을 그래프로 시각화합니다.
- **유형 분석**: 참여자들이 어떤 진단(AI 역량, 리더십 등)을 주로 이용하는지 분석합니다.
- **최근 내역**: 가장 최근에 접수된 5건의 문의를 즉시 확인 가능합니다.

### ✉️ 문의 관리 (Inquiries)
- **전체 목록**: 모든 문의 내역을 상세히 조회합니다.
- **검색**: 담당자명, 회사명, 과정명으로 특정 문의를 빠르게 찾을 수 있습니다.
- **상세 정보**: 연락처, 이메일, 희망 일정, 조직의 고민 사항 등을 확인합니다.

### 🩺 진단 결과 (Diagnostics)
- **결과 분석**: 참여자의 진단 유형과 점수를 확인합니다.
- **PDF 보기**: 시스템에서 생성된 결과 리포트 PDF를 즉시 열람할 수 있습니다.

## 3. 데이터 활용하기

### 📥 CSV 내보내기
- 각 관리 탭 우측 상단의 **[CSV 내보내기]** 버튼을 클릭하면 현재 데이터를 엑셀 파일로 다운로드할 수 있습니다.
- 주간/월간 보고서 작성 시 유용하게 활용하세요.

### 🔄 실시간 동기화
- 별도의 새로고침 없이도 신규 데이터가 유입되면 화면이 즉시 갱신됩니다.
- 상단 헤더에서 마지막 업데이트 시간을 확인할 수 있습니다.

## 4. 문제 해결 (FAQ)
- **데이터가 보이지 않아요**: 인터넷 연결을 확인하거나, Firebase 보안 규칙이 올바르게 설정되었는지 확인하세요.
- **로그인이 안 돼요**: 이메일 형식이 올바른지, 비밀번호가 6자 이상인지 확인하세요.
- **PDF가 안 열려요**: 팝업 차단이 설정되어 있는지 확인해주세요.
